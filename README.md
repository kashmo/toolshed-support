# ToolWise GitHub Pages site

A static landing page for ToolWise. No build system or dependencies are required.

## Deploy

1. Copy all files and the `assets` directory into the root of the GitHub Pages repository.
2. Commit and push to the branch configured under **Settings → Pages**.
3. Keep `index.html`, `privacy.html`, and `support.html` at the repository root to preserve the URLs.

## Main files

- `index.html` — product landing page
- `styles.css` — full responsive design
- `script.js` — reveal animations and interactive screenshot showcase
- `privacy.html` — privacy policy
- `support.html` — support and feedback page
- `assets/` — app icon and optimized screenshots
